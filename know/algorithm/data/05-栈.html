# 什么是栈

栈跟队列有些相似, 它的特点是:

  1. 先进后出
  2. 栈满还存会“上溢”，栈空再取会“下溢”

借助 js 中 pop, push可以实现一个简单的栈

# 进

Array.pop()

# 出

Array.push()

# 代码实现

let stack = []
// 只能使用以下两个 api
stack.push()
stack.pop()

在 js 的函数调用中, 都是使用了栈来保存一些中间信息。
