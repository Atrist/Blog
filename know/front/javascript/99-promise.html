# prommise

Promise 对象用于表示一个异步操作的最终完成 (或失败)及其结果值, 是一种异步编程的解决方案, 最开始由社区提出, 最后由ES6写入语法规范, 原生提供了promise对象.

promise对象有以下两个特点:

  1. 状态不受外界影响, promise一共有三个状态:

    1. pending(进行中)
    2. fulfilled(已成功)
    3. rejected(已失败)

  2. 状态一旦改变,就不会再变.

    对象状态的变化只能由 pending => fulfilled 或 pending => rejected. 状态一旦变化, 就无法再变, 此时 promise称为 resolved(已定型或已完成)
