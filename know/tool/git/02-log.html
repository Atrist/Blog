# log 命令

这里详细的罗列一些 log 常用的格式化命令, 便于更加有效率了查看项目log日志.

详细文档: https://www.git-scm.com/docs/git-log (opens new window)

# reverse

git log --reverse

以倒序的方式输出log日志

# 查看远程仓库的日志

git log 远程仓库名

示例

git log remotes/origin/master
