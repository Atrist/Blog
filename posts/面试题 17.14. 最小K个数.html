# 题目

设计一个算法，找出数组中最小的k个数。以任意顺序返回这k个数均可。

# 题解1： 排序，输出

借助 sort 进行排序之后，修改length切断数组。

/**
 * @param {number[]} arr
 * @param {number} k
 * @return {number[]}
 */
var smallestK = function(arr, k) {
    const result = arr.sort((a,b)=>a-b)
    result.length = k
    return result
};
