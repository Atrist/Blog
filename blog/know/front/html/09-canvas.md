# canvas
HTML5新增一个的元素, 借助它提供的 canvas API, 开发者可以使用JavaScript来绘制图形。它可以用于动画、游戏画面、数据可视化、图片编辑以及实时视频处理等方面。
```html
<canvas id="canvas"></canvas>
```
