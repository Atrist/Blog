## 1. Setting Sync

使用 github gits 作为文件,来云上保存 vscode 的插件目录,可以用于多终端同步你的 vscode 插件

前提条件:
需要一个 github 账号

### 快捷键

1. 上传 : Shift + Alt + U
2. 下载 : Shift + Alt + D

(on macOS: Shift + Option + U / Shift + Option + D)
